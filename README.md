# Arduino Embedded Systems

Arduino and embedded systems coursework - assessments and project.

## Exercises

- **01-led-blink** - Blink an LED on and off every second
- **02-serial** - Print to Serial on startup and in loop

## Assessments

- **01-led-blink-serial** - Blink an LED with Serial output
- **02-circuit** - Blink an LED on and off every second
- **03-pwm-led** - Set LED brightness to 200 using PWM
- **04-led-fade-pwm** - Fade LED in and out using PWM
- **05-button** - Read button state and print to Serial
- **06-button-led** - Control LED with button
- **07-button-serial-plotter** - Visualise button state in Serial Plotter
- **08-potentiometer-serial-plotter** - Visualise potentiometer value in Serial Plotter
- **09-potentiometer-led** - Control LED brightness with potentiometer
- **10-analog-pin-digital-output** - Use analog pin A5 as digital output to blink LED
- **11-serial-blink-control** - Control LED blink speed via Serial input
- **12-millis-duration** - Measure elapsed time using millis()
- **13-millis-non-blocking-serial** - Non-blocking Serial print using millis()
- **14-millis-two-led-blink** - Blink two LEDs independently using millis()
- **15-three-led-serial-potentiometer-button** - Control three LEDs with Serial, potentiometer and button
- **16-button-state-change** - Detect button press and release using state change detection
- **17-button-debounce** - Debounce button using millis()
- **18-three-led-debounce-toggle** - Blink LED 1, toggle LED 2 and 3 with debounced button
- **19-interrupt-button-led** - Toggle LED using hardware interrupt
- **20-interrupt-debounce-led** - Toggle LED using hardware interrupt with debouncing
- **21-interrupt-button-counter** - Count button presses using hardware interrupt
